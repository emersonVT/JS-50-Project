# Progress Steps

A small interactive project that demonstrates a **step progress indicator** built with **HTML, CSS, and JavaScript**.

Users can navigate through steps using **Next** and **Previous** buttons while a progress bar visually updates to reflect the current step.

##  How It Works

* The circles represent steps in the process.
* Clicking **Next** increases the current step.
* Clicking **Previous** decreases the current step.
* JavaScript dynamically:

  * Adds or removes the `active` class
  * Updates the width of the progress bar
  * Enables or disables navigation buttons
